### Somnia auto transfer every 24 hours

#### Sync Repo
```
git clone dulu
```

#### Install Modul
```
pip install web3
```

#### Set private key (increment)
```
PRIVATE_KEYS = [
    "yourprivatekey1",
    "yourprivatekey2",
]
```

#### Set address target
```
TARGET_ADDRESS = "addresstarget" 
```

#### RUN Bot
```
python3 run.py
```
