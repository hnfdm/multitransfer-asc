### 🎲 Multi Transfer EVM

A simple bot to automate multi-transfer on EVM based.

### 🚀 Features
- *Adjustable Time* 
- *Multi Account*
- *One to Many*
- *Many to One*

### 🛠️ Installation

#### 1️⃣ Clone repository
```sh
git clone https://github.com/hnfdm/multitransfer-asc.git 
```

#### 2️⃣ Enter & install dependencies
```sh
cd multitransfer-asc && pip install web3
```

#### 3️⃣ Edit **Private Key & Address** file

```sh
nano one-to-many.py
```


```sh
nano many-to-one.py
```

#### 4️⃣ Run the bot

- For One to Many
```sh
node one-to-many.py
```
- For Many to One
```sh
node many-to-one.py
```

### 🛒 Prerequisite
- Python
- Web3 Module
